# Getting Started with Troyan-3500

Welcome to the Troyan-3500 development wiki. This guide will help you get started with the project.

## Overview

Troyan-3500 is a real-time strategy game set in a sci-fi universe. This documentation covers everything from setting up your development environment to contributing code.

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (v16 or higher)
- npm (v8 or higher)
- Git

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/troyan-3500.git
   ```

2. Navigate to the project directory:
   ```bash
   cd troyan-3500
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Running the Development Server

To start the development server with hot reloading:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## Building for Production

When you're ready to build for production:

```bash
npm run build
```

The built files will be in the `dist` directory.
