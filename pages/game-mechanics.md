# Game Mechanics

This document outlines the core game mechanics of Troyan-3500.

## Core Gameplay Loop

The Troyan-3500 gameplay loop consists of:

1. **Resource Gathering** - Collect resources from the map
2. **Base Building** - Construct buildings and defenses
3. **Unit Production** - Train military units
4. **Combat** - Engage enemy forces
5. **Expansion** - Claim new territories

## Resource System

There are three primary resources in Troyan-3500:

- **Metal**: Used for constructing buildings and units
- **Energy**: Powers buildings and weapons
- **Research Points**: Used for technological upgrades

Each resource has its own gathering mechanics and storage limits.

## Combat Mechanics

Combat in Troyan-3500 features:

- **Real-time battles** with pausing capability
- **Unit types** with strengths and weaknesses
- **Terrain advantages** affecting combat outcomes
- **Special abilities** for hero units
- **Formation bonuses** for grouped units

## Technology Tree

The technology tree is divided into four branches:

1. **Military** - Improved weapons and armor
2. **Economy** - Better resource gathering and processing
3. **Defense** - Stronger fortifications and defenses
4. **Utility** - Special abilities and support functions

Each branch has multiple tiers that unlock progressively.
